About

"Monitor" is a cloud-based sensor monitoring and alert management platform which can be used to centrally monitor any device regardless of its location. The main functionality of the system is to alert on certain events based on the readings coming from the sensors. There are three sensors. those are, Temperature sensor, Humidity sensor and Pressure sensor. .If the current sensor reading goes above a certain threshold the user is able to get an notification through which notification method he likes.
